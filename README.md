# DualFlipClock Android

Android version of DualFlipClock.

## Features

- Landscape full-screen dual flip clock.
- Black background with star field.
- Screen stays awake while running.
- Two clock rows with city/school selection.
- Top row defaults to Monterey.
- Bottom row can use GPS or a selected city/school.
- Local date/time by selected city time zone.
- Night dimming after 23:00 and before 07:00 in the bottom row time zone.
- Weather refresh every 30 minutes using Open-Meteo.
- City, university, and US high-school search.
- University and high-school acronym search.
